# <Fall> <2025> Assignment 2

## Project Name: Rock Paper Scissors

## Description: The classic rock paper scissors game against the computer. User can select the number of points required to win the game by entering an integer number as an argument, followed by rock, paper or scissors. Winner gains 2 points, loser gets deducted a point, draw gives a point randomly to either user or computer to make the game interesting and adding a little twist. Enjoy!
